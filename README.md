# Beneficiary Lifecycle Management System

## Overview

This repository presents a structured operational system for beneficiary lifecycle management in a social support context.

Main functional areas:
- beneficiary intake from forms
- eligibility checks
- annual beneficiary database management
- blacklist governance
- historical tracking
- dashboarding
- fair selection workflows for activities and events

## Repository Structure

- scripts/core: core operational scripts
- scripts/selection: activity-specific selection scripts
- docs: documentation and archived material
- samples: fictitious datasets to be added manually

## Data Privacy

No real beneficiary data is included in this public version.
All operational identifiers must be replaced with local configuration values before use.
